# LOLCODE

## Instructions
1. To run, install the lolcode interpreter
```
$ git clone https://github.com/justinmeza/lci.git
$ cd lci
$ cmake .
$ make && make install
```
2. Run the using the command: ```lci main.lol``` in the ./lolcode/ directory.
